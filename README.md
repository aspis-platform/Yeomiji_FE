# (사) 여수 여미지 유기동물 보호협회

## 프로젝트 소개
여수 여미지 유기동물 보호소 웹사이트 프론트엔드 입니다.  

실제 사이트는 [여기](https://yeomiji-shelter.ncloud.sbs/)에서 확인할 수 있습니다.

## 주요 기능
- 메인 페이지에서 보호소 간단 소개
- 보호소 단체 정보
- 입양 가능한 동물들 목록
- 봉사활동 신청 방법
- 후원 계좌 정보

## 기술 스택
- ReactJS
- TS
- React Router Dom

## 커밋 규칙
- feat: 기능 추가
- chore: 환경 설정 수정
- fix: 버그 해결
- rename: 파일, 폴더명 수정 또는 이동
- remove: 파일 또는 폴더 제거
- refactor: 리팩토링
- doc: 주석/문서 추가 및 변경

[여미지 보호소 사이트 바로가기](https://yeomiji-shelter.ncloud.sbs/)
